Scala Qubytronica Estesa (Qt₁–Qt₉)

Una rotazione completa Qt₁→Qt₉ costituisce 1 SLQ, verificato contro UTC(IT) (INRiM).

  • Qt₁ 0,1s — fisico/dinamico
  • Qt₃ 0,3s — legale base
  • Qt₅ 0,5s — notariale
  • Qt₇ 0,7s — governance
  • Qt₉ 0,9s — archiviazione Ω∞

Pipeline κ/ε (fail-closed)

  1. Acquire Σ (sensori/clock)
  2. Index κ (monotonia)
  3. Estimate ε̂ (ΞΩ hybrid)
  4. Validate Χτ (fail-closed)
  5. Build Manifest (JSON canonical + prev_hash)
  6. Dual-Sign (Ed25519 + PQC opz.)
  7. Anchor (IPFS/BTC/EVM federato)

Regole di realtà (UE-first)

  • Hash-only: nessuna custodia di dati personali.
  • Fail-closed: se non verificabile → non valido.
  • Append-only: si estende, non si riscrive.
  • Audit riproducibile: verifica offline deterministica.

Questo repository descrive un sistema tecnico e un tracciato verificabile; non promette validità legale automatica senza l’inquadramento corretto e senza i requisiti normativi applicabili (eIDAS/QTSP ecc.).
