# UNEBDO — Unità Normativa di Evidenza e Decisione Operativa

UNEBDO è il **fondamento giuridico-computabile**
dell’ecosistema Hermeticum B.C.E.

Definisce **quando** un’evidenza tecnica può esistere,
non **che cosa** essa significhi giuridicamente.

## Funzione
- stabilire condizioni tecniche di validità
- supportare audit riproducibile
- impedire propagazioni non verificabili

## Principio cardine
**FAIL-CLOSED**  
Se una condizione non è verificabile, l’evidenza è invalidata.

## Non è
- una norma
- un regolamento
- un’autorità decisionale
- un sistema permissivo

## Relazioni
- OPC applica policy sopra UNEBDO
- IPR utilizza UNEBDO per la validità tecnica
- Gateway UE espone UNEBDO come fondamento
